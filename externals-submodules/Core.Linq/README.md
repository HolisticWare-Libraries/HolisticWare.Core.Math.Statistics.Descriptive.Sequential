# HolisticWare.Core.Linq

HolisticWare.Core.Linq

LINQ APIs for some uiltities (like `Even()` and `Odd()`) and for LINQ to `Memory<T>` and LINQ to
`Span<T>`

NOTE: sideproject for learning LINQ.
